# serpctl

Perform SEO evaluation on Google Search with controlled SERP parameters.

## Why use serpctl?

serpctl helps you generate clean, non-personalized Google search URLs,
making it easier to analyze SERP results without personalization bias.

## System Requirements

* A recent version of Node.js

## Usage

```sh
$ serpctl -hl zh-TW -gl tw java 教學
https://www.google.com/search?q=java+%E6%95%99%E5%AD%B8&hl=zh-TW&gl=tw&pws=0
```

The generated URL is also copied to your clipboard automatically.

## Advanced Usage

To achieve more consistent and controlled SERP results during SEO evaluation:

* Use a browser without a logged-in Google account
* Use a private (incognito) window
* Use a VPN when geographic location matters
